6 3D Графика

6.1 Графическое окно

Нажмите , чтобы открыть графическое окно (доступно только для систем размерности 2 и выше). Вы можете открыть несколько окон нажатием той же кнопки.

Здесь вы можете выбрать переменные для координатных осей, а так же вращать график при помощи движков и полей ввода на левой стороне. При нажатой левой/правой кнопке мыши вращение происходит по осям XY и XZ соответственно. Используйте поле Zoom для изменения масштаба.

6.2 Панорамирование

Нажмите кнопку для входа в этот режим (курсор мыши изменится). В этом режиме можно двигать график для просмотра его различных частей.

6.3 Свойства графика

Нажмите кнопку , чтобы открыть окно свойств:

Здесь можно:

1.
Включить/выключить изображение осей. Если оси включены, то можно:
  • Редактировать длины осей (в координатных единицах) вручную.
  • Нажать кнопку “Цвет...” для изменения цвета каждой из осей.
  • Нажать кнопку “Установить” для автоматического выравнивания длин осей под текущих график.
2.
Установить ширину линий для осей и решений. Иногда для лучшего качества изображений имеет смысл установить ширину вручную.
3.
Установить диаметр маркеров решений (в пикселах).

6.4 Экспорт графика в файл

Нажмите кнопку для открытия окна экспорта:

Здесь можно:

1.
Выбрать тип изображения:

Этот список зависит от платформы.

2.
Задать ширину и высоту будущего изображения в пикселах.
3.
Установить качество кодирования JPEG.
4.
Открывать созданное изображение во внешней программе, которая установлена по умолчанию на компьютере.

6.5 Справка

Нажмите кнопку , чтобы открыть данное руководство.
