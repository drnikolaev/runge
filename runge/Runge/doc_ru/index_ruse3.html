3 Главное меню

3.1 Меню Файл

Это меню позволяет:

  • Создать новый файл (новую систему ОДУ для решения) используя комбинацию Ctrl N.
  • Открыть существующий файл Рунге используя комбинацию Ctrl O.
  • Открыть ранее сохранённый файл при помощи под-меню

  • Сохранить текущий файл используя комбинацию Ctrl S.
  • Сохранить текущий файл под другим именем.
  • Выйти из Рунге.

Файл Рунге хранит систему, все параметры, все решения и их свойства.

3.2 Меню Настройки

Это меню позволяет:

  • Выбрать конфигурационный файл Рунге. Используйте это только в том случае, когда требуется добавить свой собственный алгоритм. Используйте стандартный файл Runge*.xml в противном случае.
  • Выбрать шрифт для программирования выражений (моноширинные шрифты семейства Courier улучшают читаемость).
  • Выберите язык (требуется перезагрузка программы в случае смены языка). В настоящее время поддерживаются русский и английский языки.

3.3 Меню Старт

Это меню позволяет:

  • Начать решение системы.
  • Приостановить, если решение начато.
  • Продолжить, если решеие приостановлено.
  • Остановить, если решение начато.

3.4 Меню График

Открывает окно с 2D или 3D графиками решений (можно открыть несколько окон).

3.5 Меню Справка

Открывает справочное руководство по Рунге.
